# solid-pixel

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

a pixel renderer

## Quick start

Install it:

```bash
npm i solid-pixel
# or
yarn add solid-pixel
# or
pnpm add solid-pixel
```

Use it:

```tsx
import solid-pixel from 'solid-pixel'
```
